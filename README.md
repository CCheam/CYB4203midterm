This HTML application demonstrates different attacks in the AI/ML lifecycle in a military context. Simply drag/drop the attack vectors to see the effect in the targeting view. No dependencies, just download and run the hmtl in the browser. All the attacks are combined into a singular artifact for ease of interaction.            
Data poisoning marks clearly civilian targets as hostile.      
Weight manipulation (malicious weights) marks mixed use targets as hostile.       
Adversarial input marks enemy hostiles as civilian in nature.
