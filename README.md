This HTML application demonstrates different attacks in the AI/ML lifecycle in a military context. Simply drag/drop the attack vectors to see the effect in the targeting view.   
The attack vectors and a model AI/ML lifecycle to drag the vectors into are on the left hand side and a 'target area' is generated on the right hand side, with a mix of civilian/hostile targets.        
No dependencies, just download and run the hmtl in the browser. 3 different attacks are represented in this artifact for ease of interaction.        
________________________________________________________________________________________________________________
Data poisoning marks clearly civilian targets as hostile.      
Weight manipulation (malicious weights) marks civilian targets as hostile based on a demographic flag.       
Adversarial input marks enemy hostiles as civilian in nature.       
______________________________________________________________________________________________________________
This simulator is named SeeTac after the Seattle Airport. Seattle is foggy, fog of war, thus SeeTac Targeting System.
