This HTML application demonstrates different attacks in the AI/ML lifecycle in a military context. Simply drag/drop the attack vectors to see the effect in the targeting view. No dependencies, just download and run the hmtl in the browser.  <\br>
Data poisoning marks clearly civilian targets as hostile.  <\br>
Weight manipulation (malicious weights) marks mixed use targets as hostile.   <\br>
Adversarial input marks enemy hostiles as civilian in nature.   <\br>
